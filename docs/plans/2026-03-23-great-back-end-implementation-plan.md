# Great Back End — 10-Day Implementation Plan

## Day 1-2
- Bootstrap app skeleton in code repo
- Define RBAC roles: owner / partner
- Create data contracts for editable copy and dashboard config

## Day 3-4
- Chat page + conversation create/history APIs
- Gateway proxy for partner instance
- Basic audit log pipeline

## Day 5-6
- Dashboard widgets: token usage / tasks / health
- Copy editor UI and write-back export to data repo format

## Day 7
- Logical isolation rollout (`lighthouse` user + dedicated OpenClaw instance)
- Access policy validation

## Day 8
- Push notifications (web push first)
- iPhone installability and UX polish

## Day 9
- End-to-end hardening, bug fixes, demo script

## Day 10
- Final rehearsal + delivery package

## Delegation protocol
- Main agent owns architecture and final integration.
- Minimax handles parallel tasks: benchmark research, UI copy drafts, test case expansion, docs linting.
- Every delegated task must return verifiable outputs (files/tests/commands).
